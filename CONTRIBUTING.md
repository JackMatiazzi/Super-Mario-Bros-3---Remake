# Como Contribuir

Não commite direto na `main` — a branch está protegida e vai rejeitar.
Crie uma branch, faça o PR e peça pra um dos colegas aprovar antes de mergear.

## Criando uma branch

```
git checkout -b feature/nome-da-tarefa
```

## Convenção de commits

| Prefixo | Quando usar |
|---|---|
| `feat:` | nova mecânica ou sistema |
| `fix:` | correção de bug |
| `level:` | criação ou edição de fase |
| `art:` | sprites, animações, visual |
| `audio:` | sons e músicas |
| `docs:` | README, comentários |
| `chore:` | configuração, setup |
